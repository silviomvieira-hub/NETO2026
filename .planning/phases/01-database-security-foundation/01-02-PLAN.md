**1. Criar `supabase/rls-policies.sql`:**

Per D-13: RLS policies em TODAS as tabelas. Per D-15: usar `auth.uid()` e profiles table join.

Estrategia de RLS:
- Funcao helper `get_user_role()` que busca role da tabela profiles
- Funcao helper `get_user_tenant_id()` que busca tenant_id da tabela profiles
- Funcao helper `get_user_regions()` que busca regions array da tabela profiles
- Tenant isolation: TODA query filtrada por tenant_id
- Admin: ve tudo do tenant
- Coordenador: ve dados das suas regioes (regions array) + seus proprios registros
- Cabo eleitoral: ve apenas seus proprios registros

```sql
-- =============================================================
-- CampanhaApp RLS Policies v1
-- Per D-13: RLS on ALL tables
-- Per D-15: auth.uid() + profiles table for role/tenant
-- EXECUTE AFTER schema.sql
-- =============================================================

-- ===== HELPER FUNCTIONS =====

-- Get current user's tenant_id from profiles
CREATE OR REPLACE FUNCTION get_my_tenant_id()
RETURNS UUID AS $$
  SELECT tenant_id FROM profiles WHERE id = auth.uid()
$$ LANGUAGE sql SECURITY DEFINER STABLE;

-- Get current user's role from profiles
CREATE OR REPLACE FUNCTION get_my_role()
RETURNS TEXT AS $$
  SELECT role FROM profiles WHERE id = auth.uid()
$$ LANGUAGE sql SECURITY DEFINER STABLE;

-- Get current user's assigned regions
CREATE OR REPLACE FUNCTION get_my_regions()
RETURNS UUID[] AS $$
  SELECT COALESCE(regions, '{}') FROM profiles WHERE id = auth.uid()
$$ LANGUAGE sql SECURITY DEFINER STABLE;

-- ===== ENABLE RLS ON ALL TABLES =====
ALTER TABLE tenants ENABLE ROW LEVEL SECURITY;
ALTER TABLE regions ENABLE ROW LEVEL SECURITY;
ALTER TABLE profiles ENABLE ROW LEVEL SECURITY;
ALTER TABLE eventos ENABLE ROW LEVEL SECURITY;
ALTER TABLE pessoas ENABLE ROW LEVEL SECURITY;
ALTER TABLE lideres ENABLE ROW LEVEL SECURITY;
ALTER TABLE quem_e_quem ENABLE ROW LEVEL SECURITY;
ALTER TABLE demandas ENABLE ROW LEVEL SECURITY;
ALTER TABLE pesquisas ENABLE ROW LEVEL SECURITY;
ALTER TABLE materiais ENABLE ROW LEVEL SECURITY;
ALTER TABLE visitas ENABLE ROW LEVEL SECURITY;
ALTER TABLE atividades ENABLE ROW LEVEL SECURITY;
ALTER TABLE gamificacao_xp ENABLE ROW LEVEL SECURITY;
ALTER TABLE tarefas_game ENABLE ROW LEVEL SECURITY;
ALTER TABLE streaks ENABLE ROW LEVEL SECURITY;
ALTER TABLE trofeus ENABLE ROW LEVEL SECURITY;

-- ===== TENANTS POLICIES =====
-- Users can only see their own tenant
CREATE POLICY "tenants_select" ON tenants
  FOR SELECT USING (id = get_my_tenant_id());

-- ===== REGIONS POLICIES =====
```
