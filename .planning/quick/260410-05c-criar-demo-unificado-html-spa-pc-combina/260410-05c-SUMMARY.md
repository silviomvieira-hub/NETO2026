## Technical Decisions

- **Single File SPA**: Portabilidade máxima, distribuição simples para cliente
- **Hash Routing (#)**: Funciona sem server config, abre direto como arquivo local
- **CSS Variables**: 8 cores temáticas centralizadas, fácil manutenção
- **Glass Morphism**: Estética moderna (blur + rgba surfaces)
- **SVG Charts**: Termômetro (gauge circles), Sentimento (line chart com gradient) — sem Chart.js
- **Mock Data**: Estruturas realistas preparadas para migração Supabase
- **Unicode Icons**: Lightweight (evita sprite SVG), funciona em qualquer navegador
- **20 Animações CSS**: Pure CSS keyframes, full control, sem animation library
- **Auto-update Feed**: setInterval 8s com fade animations para sensação de "ao vivo"
