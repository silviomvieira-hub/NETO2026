## Technical Decisions

- **Single File SPA**: Maximizes portability and ease of distribution to client
- **Hash Routing**: Works without server configuration, perfect for local file opening
- **CSS Grid**: Responsive without media queries for desktop-first layout
- **Inline CSS/JS**: Eliminates build step, no Node.js required
- **Mock Data**: Realistic data structures prepared for future Supabase migration
- **Glass Morphism**: Modern aesthetic aligned with product vision
- **Unicode Icons**: Lightweight alternative to SVG sprite, works everywhere
- **Keyframe Animations**: Pure CSS, no animation library, full control and performance
