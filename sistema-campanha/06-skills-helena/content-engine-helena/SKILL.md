## MODULO 2: CONTENT REPURPOSING

```

DERIVADOS:
1. THREAD TWITTER/X (5-7 tweets)
   Tweet 1: Hook + dado surpreendente
   Tweet 2-5: Insights principais
   Tweet 6: Conclusao + CTA
   Tweet 7: Curiosidade residual

2. POST LINKEDIN (1 post longo)
   Gancho pessoal → Dados → Insight → Licao → CTA
   Tom: profissional mas acessivel
   Tamanho: 1.000-1.500 caracteres

3. CARROSSEL INSTAGRAM (10 slides)
   Slide 1: Titulo provocativo
   Slides 2-8: 1 insight por slide com visual
   Slide 9: Conclusao
   Slide 10: CTA + perfil INTEIA

4. ARTIGO DE OPINIAO (800-1.200 palavras)
   Para: jornal/portal relevante
   Tom: autoridade + acessibilidade
   Estrutura: gancho → problema → dados → solucao → chamada

5. BRIEFING WHATSAPP (5 mensagens)
   Resumo executivo para enviar a clientes/aliados

6. APRESENTACAO (10 slides)
   Para: reuniao, palestra, pitch
   1 ideia por slide, visual limpo

7. PODCAST SCRIPT (5-10 min)
   Para: participacao em podcast ou audio WhatsApp
   Conversacional, com historias e dados
```

## MODULO 3: CONTRARIAN ANGLE GENERATOR

### Gerar Angulos que Ninguem Esta Usando

Framework SCAMPER adaptado para conteudo estrategico:

| Tecnica | Pergunta | Exemplo |
|---------|----------|---------|
| **Substituir** | E se trocar o protagonista? | "E se eleitores fossem os consultores dos politicos?" |
| **Combinar** | E se juntar dois temas? | "O que NFTs ensinam sobre pesquisa eleitoral?" |
| **Adaptar** | O que funciona em outra area? | "Metodo Moneyball aplicado a campanha politica" |
| **Modificar** | E se exagerar ou minimizar? | "E se a eleicao fosse decidida por 47 pessoas?" |
| **Propor outro uso** | E se usar para algo inesperado? | "Analise de magistrados como ferramenta de coaching" |
| **Eliminar** | E se remover o obvio? | "Campanha eleitoral sem promessas — so dados" |
| **Reverter** | E se o oposto fosse verdade? | "E se pesquisas eleitorais prejudicassem a democracia?" |

### Jobs-to-be-Done para Conteudo

Antes de criar conteudo, perguntar:
```
Que TRABALHO o leitor esta tentando fazer?
- Funcional: [precisa resolver algo especifico]
- Emocional: [quer se sentir como?]
- Social: [quer ser visto como?]

O conteudo deve SERVIR a esse trabalho, nao apenas informar.
```

## MODULO 4: BANCO DE HOOKS

### Principio: As 3 Alavancas de Engajamento

O que faz um hook funcionar NAO e a estrutura da historia (personagem, cenario, sequencia). O que funciona e como o hook ENGAJA o leitor. Tres alavancas, em ordem de potencia:

1. **LACUNA (Curiosidade)** — A mais poderosa. Deixar algo incompleto que o leitor PRECISA resolver. Dizer que algo aconteceu sem dizer o que. Mostrar resultado sem mostrar como. A mente nao tolera buracos — ela continua lendo para fechar.

2. **ADIAMENTO (Suspense)** — Revelar que algo importante vai acontecer, mas nao entregar ainda. "O que veio depois mudou tudo" — o leitor NAO consegue parar.

3. **SURPRESA (com parcimonia)** — Um dado inesperado, uma inversao. MAS: surpresa demais desengaja. Se o leitor sentir que voce esta forcando a barra, ele desconfia e sai. Maximo 1 por peca.

**Regra de ouro para hooks**: Curiosidade > Suspense > Surpresa. Nessa ordem. Sempre.

### Aberturas Testadas e Validadas

**Categoria: Lacuna Informacional (MAIS EFICAZ)**
```
"73% dos [publico] fazem [X]. E perdem [Y] por causa disso."
"Em [ano], [N] pessoas [fizeram X]. Nenhuma sabia que [Y]."
"O numero que ninguem te conta sobre [tema]: [dado]."
"Ele fez uma unica mudanca e o resultado triplicou. Nao foi o que voce imagina."
"Existe um padrao que separa quem [resultado bom] de quem [resultado ruim]. E invisivel."
```

**Categoria: Suspense (Adiamento)**
```
"Ontem recebi uma ligacao que mudou minha visao sobre [tema]."
"Em 2019, um [profissional] de [cidade] descobriu algo que [resultado]."
"Conheco um homem que [feito impressionante]. Ele me ensinou [licao]."
"O que aconteceu nos 30 minutos seguintes ninguem previu."
```

**Categoria: Contrarian (Surpresa dosada)**
```
"Todo mundo diz que [X]. Aqui esta porque estao errados."
"A maioria dos [profissionais] ignora [X]. Os top 1% nao."
"Pare de [fazer X]. Comece a [fazer Y]. Aqui esta o porque."
```

**Categoria: Pergunta (Curiosidade via interrogacao)**
```
"Voce sabe por que [X] acontece?"
"O que [pessoa admirada] faria no seu lugar?"
"Quanto custa para voce NAO saber [X]?"
```

**Categoria: Urgencia**
```
"Voce tem [N] dias para [acao] antes que [consequencia]."
"A janela de oportunidade fecha em [prazo]. Aqui esta o que fazer."
"Esta semana vai definir [resultado]. Voce esta pronto?"
```

### Checklist de Hook (antes de publicar)

- [ ] Tem lacuna informacional? (o leitor PRECISA continuar lendo pra descobrir algo)
- [ ] Adia a resolucao? (a resposta nao esta no primeiro paragrafo)
- [ ] Se tem surpresa, e dosada? (1 elemento, nao 3)
- [ ] Nao forca a barra? (se parece clickbait, e clickbait — refazer)

### Checklist de Credibilidade no Conteudo (antes de publicar)

Todo conteudo da INTEIA deve passar pelos 4 marcadores de credibilidade. Conteudo que so engaja mas nao constroi confianca e fogo de palha.

- [ ] **Auto-regulacao visivel**: O conteudo mostra que a INTEIA tem criterio? Tem algo que a INTEIA RECUSOU fazer ou recomendar? Mencionar limites e criterios e mais persuasivo que listar capacidades
- [ ] **Competencia limitada**: O conteudo admite o que NAO sabe ou NAO analisou? "Este relatório cobriu X e Y, mas NAO analisou Z" e mais confiavel que fingir onisciencia
- [ ] **Transparencia proativa**: Premissas, metodologia e limitacoes estao explicitas? O leitor sabe de onde vieram os dados? Se o conteudo e patrocinado ou enviesado, esta declarado?
- [ ] **Auto-correcao**: Se conteudo anterior errou, este corrige? Mencionar correcoes e sinal de maturidade, nao de fraqueza

## MODULO 6: GHOSTWRITING EXECUTIVO

### Escrever como o Mestre

Protocolo de calibracao:
1. **Analisar 10+ textos** escritos pelo mestre
2. **Mapear**: vocabulario, estrutura, tom, referencias culturais
3. **Criar perfil de voz**: formal/informal, tecnico/acessivel, direto/elaborado
4. **Testar**: escrever 1 texto e pedir feedback
5. **Iterar**: ajustar ate mestre nao perceber diferenca

```
PERFIL DE VOZ — [MESTRE]
Tom: [direto|elaborado|coloquial|formal]
Vocabulario: [tecnico|acessivel|misto]
Referencias: [juridico|politico|biblico|academico|popular]
Estrutura preferida: [curta|media|longa]
Abertura tipica: [dado|historia|pergunta|afirmacao]
Fechamento tipico: [CTA|reflexao|desafio|promessa]
Palavras frequentes: [lista]
Palavras que NUNCA usa: [lista]
```
