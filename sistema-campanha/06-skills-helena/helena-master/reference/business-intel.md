# Business Intelligence — Referencia Detalhada

## Viabilidade Financeira (Pilar 4 Detalhado)

| Metrica | Calculo |
|---------|---------|
| Investimento inicial | CAPEX (equipamentos, reforma, licencas) |
| Custo fixo mensal | Aluguel + pessoal + utilidades + marketing |
| Custo variavel | Insumos + comissoes + frete |
| Ticket medio | Pesquisa de mercado + posicionamento |
| Break-even | Custo fixo / (Ticket - CV unitario) |
| Payback | Investimento / Lucro liquido mensal |
